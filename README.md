Projeto prático de simulação de ataques de força bruta utilizando Kali Linux e Medusa em ambiente isolado (VirtualBox host-only). Foram testados três cenários: força bruta em FTP, ataque a formulário web via DVWA e password spraying em SMB com enumeração de usuários via enum4linux. O projeto inclui wordlists personalizadas, comandos executados, resultados obtidos e recomendações de mitigação para cada vetor de ataque explorado.
